Description

Welcome to the Simple Fractal Generator! This tool lets you create custom fractals by adjusting parameters such as sides, depth, scale, spread, and branches. Customize these settings and then click "Apply Settings" to generate your fractal pattern on the canvas.


Color Gradient: Choose two colors to set the gradient that will be used in the fractal's design.


Sides: Defines the number of sides for the fractal pattern. For example, a setting of 4 will generate a four-sided pattern.


Depth: Sets the level of recursion, or depth, in the fractal pattern. Higher depths create more intricate designs but may take longer to render.


Scale: Controls the scaling factor for each recursion level. Smaller values create tighter, more compact designs.


Spread: Adjusts the spread or angle of branches in the fractal. Experiment with this to see different effects.


Branches: Specifies the number of branches from each point in the fractal. Higher values create more complex branching structures.


Experiment with different settings to create unique and intricate fractal designs. Enjoy exploring!
